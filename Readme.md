# 🚀 DebtFlow API

API desenvolvida com Node.js e Express como parte do meu aprendizado em desenvolvimento Full Stack.

O objetivo deste projeto é evoluir para um sistema de negociação de dívidas, começando pela construção de uma API de gerenciamento de clientes.

---

## 🧠 Sobre o projeto

Este projeto está sendo construído do zero com foco em entender na prática como funciona o backend:

- Criação de servidor com Node.js
- Estruturação de rotas
- Manipulação de requisições e respostas (req / res)
- Uso de JSON no backend
- Armazenamento de dados em memória

---

## ⚙️ Tecnologias utilizadas

- Node.js
- Express
- TypeScript
- ts-node-dev

---

## 📌 Funcionalidades atuais

- ✔ Criar cliente (POST /clientes)
- ✔ Listar clientes (GET /clientes)
- ✔ Deletar cliente (DELETE /clientes/:id)
- ✔ Identificação de cliente por ID
